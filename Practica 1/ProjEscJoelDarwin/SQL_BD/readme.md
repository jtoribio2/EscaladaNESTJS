# SQL_BD
Son els  scripts SQL que s'utilitzen en  el projecte:

- `Escalada-schema.sql` — Conté l'estructura de la base de dades(DDL): creació de taules, claus primàries i foranes.  
  
- `Escalada-data.sql` — Conté les dades inicials per posa dades a  la base de dades.

Per inicialitzar la base de dades, executar primer `Escalada-schema.sql` i després `Escalada-data.sql`.
